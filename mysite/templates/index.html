What Is a Algorithm

An algorithm is the set of instructions a computer goes through to meet a goal. For example, if you have a robot and you want it to pick up a coin off the table, you first would tell it to look for a coin on the table. If it finds one, then it would proceed to the next step and so on.

or the definition that google give you: a process or set of rules to be followed in calculations or other problem-solving operations, especially by a computer.
